# Gps Wolfox KSA 🇸🇦

أداة متقدمة لتغيير الموقع الجغرافي (GPS Spoofing) لنظام iOS مع واجهة مستخدم متكاملة وتجاوز للحماية.

## 🚀 كيفية البناء (Build)

بما أنك لا تملك جهاز Mac، يمكنك البناء باستخدام **GitHub Actions** مجاناً:

1.  قم بإنشاء مستودع جديد على GitHub باسم `GpsWolfoxKSA`.
2.  ارفع كافة ملفات هذا المجلد إليه.
3.  اذهب إلى تبويب **Actions** في مستودعك.
4.  ستجد عملية بناء جارية باسم `Build KSA Tweak`.
5.  بعد انتهائها (حوالي دقيقتين)، ستجد ملف الـ `.deb` جاهزاً للتحميل في قسم **Artifacts**.

## 📦 كيفية التثبيت (Installation)

بعد تحميل ملف الـ `.deb`:
1.  قم بنقل الملف إلى جهاز الآيفون الخاص بك (عبر AirDrop أو Filza).
2.  افتح الملف باستخدام تطبيق **Filza File Manager**.
3.  اضغط على الملف واختر **Install**.
4.  بعد انتهاء التثبيت، قم بعمل **Respring** للجهاز.
5.  ستظهر لك واجهة الأداة فوق التطبيقات، أو يمكنك تفعيلها عبر الإيماءة المخصصة (النقر المزدوج أو الثلاثي حسب الإعداد).

## 🛠 المكونات التقنية
*   **اللغة:** Objective-C++
*   **المعمارية المدعومة:** arm64, arm64e (A12+)
*   **التوافق:** iOS 13.0 وما فوق.
